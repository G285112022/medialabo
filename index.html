「Webコンテンツ作成」の成果

本テーマの学習目的

先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得することが目的です。到達目標は次の4つです。(1) 手順書を読み、実験を行い、レポートやプレゼンテーションにより報告するという実験の一連の流れを実践できる、（2）コミュニケーション能力の重要性を理解し、グループワークを円滑に実施できる、（3）モーション解析・編集ができる、（4）HTML等を用いたWebページ作成ができる。なお、本授業は、リーダーシップ能力、課題に対する解決能力育成に寄与する科目です。

毎回の実験で作成したページ

第1回 WWW と HTML

今回は、Webページの特徴についてや、HTMLについての詳細を学びました。普段使っているWebページに使われている要素の種類や、HTMLの特徴、メリットやデメリットなどを学び、実際に演習で自分の力で作ってみたりとさまざまなWebページの基礎基本を学んだ。今回行った演習は、要素を使った簡単なWebページの作成や要素の使い方を学ぶ演習などである。

演習の成果

  • 演習1-3:HTML文書の作成
  • 演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回は、Webページの装飾について、CSSという要素を学んだ。CSSの基盤やスタイルシートなどの概要から、文字の色付け、フォントの変更、点線や実線の表示方法など、細かな部分まで学んだり、演習を通して見た目の変更などの重要さを、講義を通して学習した。

演習の成果

    • 第3回 JavaScript の基本

    今回は、JavaScriptというものの基本的な座学や構文、デバック方法など様々な概要について学んだ。JavaScriptの使い方、何ができるのか、htmlへの追加方法を学び、javaで学んだfor文や配列など、Webサイトを作る上で便利である様々な構文を演習を通して学習した。

    演習の成果

    • 第4回 DOM と JavaScript

    今回は、DOM(Document Object Model)という、コンテンツに変更を加えたり、内容を読み取ったり、動きのあるページにするための技術を学んだ。htmlにある要素を探し出して呼び出したり、要素を変更したりなど、直接書き換えなくても変更処理ができたり、第5回で学ぶイベント処理と組み合わせて使用することができるなど、DOMに関する知識を演習を通して学んだ。

    演習の成果

    • 第5回 イベント処理

    今回は、Webページ上で起こった動作や出来事のことを指す「イベント」について学んだ。クリックやフォーカス、キーの押し離しなどでイベントを起こすことができ、Webページの飾りにもなり実用性もあるイベントを、今回の演習全体を通して学んだ。
